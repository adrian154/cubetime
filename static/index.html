About

CubeTime is a minimalist timer app built for speedcubers.

If you are interested in contributing to the development of CubeTime, check out our GitHub repository.

Did you discover a bug, or is there a feature you would like to see added to CubeTime? Please create an issue on GitHub.

Usage Guide

To start the timer, simply hold the spacebar until the timer turns green. Then, begin solving. Press any key to stop the timer once you've finished your attempt. This interface is modeled after the StackMat timer that is used at official competitions.

CubeTime makes it easy for you to keep track of all your times across devices. Click the Session tab to see a list of times, as well as statistics and time trends.

Settings

Timer

Use inspection time

When enabled, there will be a 15-second inspection countdown before the timer begins.
Hold down time

How long the spacebar needs to be held for the timer to turn green. Use a value of 0.55 to simulate the Stackmat delay.
Play voice alerts

Voice warnings will be played when 8 and 12 seconds of inspection time have elapsed.
